# Finance Tracker V1

## Description
A simple JavaScript finance tracker that:
- Stores income
- Stores expenses as objects
- Calculates remaining balance
- Warns when balance is below 10%

## What I Improved
- Object manipulation
- Array iteration
- Cumulative subtraction
- Avoiding object reference bugs
- Pay attention to code even when writing it on paper
